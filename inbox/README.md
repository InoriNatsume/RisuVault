# Inbox (임시 보관소)

RisuAI에서 내보낸 봇/모듈/프리셋 파일을 여기에 떨어뜨리세요.
AI 에이전트가 `risuvault-add-bot` 스킬로 vault에 편입합니다.

## 지원 포맷
- 봇: `.charx`, `.png`, `.jpg`, `.jpeg`
- 모듈: `.risum`
- 프리셋: `.risup`, `.risupreset`

## 주의
- 이 폴더는 git 추적 대상이 **아닙니다** (`.gitignore`에 포함).
- vault 편입 후 원본 파일은 이 폴더에 남아있어도 되고, 안전하게 삭제해도 됩니다.
